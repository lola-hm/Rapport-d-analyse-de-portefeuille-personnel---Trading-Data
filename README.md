# Analyse de performance de portefeuille personnel – Trading Data

## Objectif
Ce projet consiste à analyser les performances de mon portefeuille de trading afin d’identifier les types d’ordres, les horaires et les stratégies les plus rentables.  

## Données utilisées
- Export des trades depuis mon compte FTMO au format CSV  
- Variables : type d’ordre, date et heure, taille du lot, durée de conservation, profit/perte

## Méthodes et outils
- Python (pandas pour la manipulation de données)  
- Visualisation avec matplotlib et seaborn  
- Export du notebook en HTML pour consultation facile

## Analyses réalisées
- Identification du type d’ordre le plus rentable  
- Détermination des jours de la semaine avec le meilleur rendement  
- Rentabilité en fonction de la taille du lot  
- Analyse du profit moyen selon la durée de conservation des trades  
- Identification des horaires de trading les plus rentables

## Résultats
- Un rapport PDF est disponible 
- Les visualisations sont disponibles dans le notebook Jupyter et le fichier HTML

## Compétences développées
- Analyse et visualisation de données réelles  
- Manipulation de fichiers CSV et Jupyter Notebook  
- Création de graphiques et interprétation de résultats pour la prise de décision
